## ✨ 魅力的な機能

### 🖼️ 推し画像表示
- **統合された表示モード制御**: 1つのボタンでランダム、サイクル、固定（ピン留め）をスマートに切り替え。
- **インテリジェントな記憶ロジック**: 固定モードを解除した際、最後に設定していた切り替え間隔を自動で復元。
- **クイック間隔設定**: 画像エリアのメニューから10秒〜1時間、または毎日決まった時間に切り替える設定が可能。
- ドラッグによる配置変更とリサイズ、クリップボードからの直接貼り付けに対応。
- **直感的な操作**: ドラッグ＆ドロップやスプリッターを使用して、画像やカレンダーの配置を自由に変更できます。

### 🎨 あなた色のカレンダー
- **カスタム休日設定**: 「推し」の誕生日やデビュー日、大切な記念日を特別な色で彩ります。
- **ダークモード対応**: 夜の推し活も快適に。OSの設定に合わせて、美しく落ち着いたテーマに切り替わります。
- **ガラスモーフィズム**: 最新のデザイントレンドを取り入れた、透明感のあるインターフェース。

### 🛠️ 自由自在な表示
- **マルチ月表示**: 1ヶ月から最大2ヶ月先まで。先の予定も一目瞭然。
- **ダイレクトなレイアウト切替**: メイン画面上のボタンからダイレクトにカレンダーの並びを縦・横に切り替えられます。
- **PWA対応**: PCやスマホのホーム画面に追加して、アプリ感覚ですぐに起動。

### 🛡️ 安心のデータ管理
- **一括バックアップ・復元**: 全設定（推し情報・記念日・タグ・表示設定など）と登録画像を1つのファイル（Gzip圧縮JSON）として書き出し・読み込みが可能。
- **画像タグパッケージ**: 画像とタグ情報のみを抽出してエクスポートし、別の環境に引き継ぐことができます。

### 📉 ストレージ最適化
- **画像自動圧縮**: 「標準」と「積極的」の2つの圧縮モードを搭載。保存時に自動的にリサイズ・品質調整を行い、ブラウザのストレージ容量を節約します。
- **一括圧縮機能**: すでに登録済みの画像に対しても、後から一括で圧縮処理を適用して空き容量を確保できます。
- **容量インジケーター**: 現在のストレージ使用状況を視覚的に把握。

### ⚡ パフォーマンスとアクセシビリティ
- **高速なカレンダー描画**: 記念日イベントの事前グルーピングと描画ループの最適化により、カレンダーの切り替えや月移動がスムーズに動作します。
- **キーボード操作への対応**: 設定画面を含む主要UIがキーボードフォーカスに対応し、アクセシビリティが向上しました。

## 🚀 今後のロードマップ
おしこよは今後もさらに使いやすく進化していきます。現在、以下の機能追加や改善を予定・検討しています。

- **モバイル表示の最適化**: スマートフォンなど画面の狭いデバイスでも快適に閲覧できるレイアウトの改善（「Web Share Target」等による連携強化済み）。
- **ダークモード切替UIの実装**: 画面上から直接ライト/ダークモードを切り替えられるスイッチの追加。

- **その他UI/UXの向上**: 4ヶ月以上の表示対応、カレンダーの過去/未来のスクロール範囲調整、印刷用CSSの最適化など。

## 📖 使い方 (Quick Start)

1. **アプリを開く**: 上記のリンクからアクセス。
2. **自分好みに設定**: 右上の ⚙️ アイコンから、画像の表示方法やカレンダーのレイアウトをカスタマイズ。
3. **記念日を登録**: あなただけの「推し事」スケジュールをカレンダーに反映。
