# Oshikoyo

お仕事（または「推し事」）のスケジュールを管理するための、美しく機能的なカレンダーアプリケーションです。

## 特徴

- **モダンなデザイン**: ガラスモーフィズム（Glassmorphism）を採用した、洗練されたインターフェース。
- **祝日対応**: 日本の祝日を自動的に表示（振替休日含む）。
- **カスタマイズ性**:
  - 週の開始日（日曜日/月曜日）を選択可能。
  - 表示月数（1〜3ヶ月）の切り替え。
  - レイアウト（横並び/縦並び）の選択。
- **PWA対応**: デスクトップやスマートフォンにインストールして利用可能。
- **ダークモード対応**: OSの設定に合わせて自動的にカラーテーマが切り替わります。

## 使い方

1. `index.html` をブラウザで開きます。
2. 右上のギアアイコンから設定を変更できます。
3. 日付表示（ヘッダー中央）をクリックすると、現在の月にリセットされます。
