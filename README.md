## ✨ 魅力的な機能

### 🖼️ 推し画像表示 & スマートレイアウト
- **統合された表示モード**: 1つのボタンでランダム、サイクル、固定（ピン留め）をスマートに切り替え。
- **インテリジェントな記憶ロジック**: 固定モードを解除した際、最後に設定していた切り替え間隔を自動で復元。
- **クイック間隔設定**: 画像エリアのメニューから10秒〜1時間、または特定時刻（0時/4時）に切り替える設定が可能。
- **直感的な操作**: ドラッグ＆ドロップやスプリッターを使用して、画像やカレンダーの配置・サイズを自由に変更できます。
- **没入モード**: 画像をフルスクリーン表示し、操作UIをマウスオーバー時のみ表示するシアター風モード。
- **スマート自動レイアウト**: 縦長画像は左配置、横長画像は上配置に自動で切り替え。手動固定も可能。
- **デフォルト画像**: 初回起動時に横長・縦長のサンプル画像が自動登録され、すぐにレイアウトの動きを体感できます。

### 🎨 あなた色のカレンダー
- **カスタム休日設定**: 「推し」の誕生日やデビュー日、大切な記念日を個別のカラーで彩ります。
- **記念日優先表示**: 記念日には、その「推し」のタグが付いた画像を優先的（または専有）に表示。
- **ダークモード対応**: 夜の推し活も快適に。OS設定に合わせ、ガラスモーフィズムを採用した透明感のあるUIに切り替わります。

### 🛠️ 自由自在な表示
- **マルチ月表示**: 1ヶ月から最大2ヶ月先まで。先の予定も一目瞭然。
- **ダイレクト切替**: メイン画面上のボタンからカレンダーの並びを縦・横に即座に切り替えられます。
- **PWA対応**: PCやスマホのホーム画面に追加して、アプリ感覚ですぐに起動。

### 🏷️ 高度な管理機能
- **画像・推しタグ管理**: 画像や推しにタグを付与し、記念日との連動や高度な分類が可能です。
- **イベントタイプのカスタマイズ**: 誕生日・デビュー記念日に加え、独自のイベントタイプをアイコン付きで作成できます。
- **CSVインポート**: 大量な記念日や推し情報をCSVから一括登録。 [テンプレートこちら](./docs/vspo_jp_sample.csv)
- **一括バックアップ・復元**: 全設定と登録画像を Gzip 圧縮 JSON 1ファイルで書き出し・読み込みが可能。
- **画像+タグパッケージ**: 画像とタグ情報だけを個別に書き出し・取り込み（既存データを保持したままマージ）。

### 📱 モバイル専用UI
- **ボトムタブナビゲーション**: ホーム・画像ライブラリ・推し管理・設定を下部タブで切り替え。
- **再生モードポップオーバー**: ホームタブをタップすると表示モードと切り替え間隔をその場で変更できます。固定モード時はホームタブにピンアイコンが表示されます。
- **推し管理パネル**: 仮想スクロール・かな行グループ表示・インデックスレールで大量の推しも快適に管理。
- **ボトムシート**: 日付をタップすると予定や記念日の詳細がボトムシートでスライドアップ表示。
- **モバイル設定パネル**: 全般・記念日・画像・イベント・データ・アプリ情報の6サブパネルで整理。

### 📉 ストレージ最適化
- **画像自動圧縮**: 保存時に自動リサイズ（標準/積極的）を行い、ブラウザのストレージ容量を節約。
- **一括圧縮機能**: 登録済みの画像に対しても、後から一括で圧縮処理を適用して空き容量を確保。
- **容量インジケーター**: ストレージ使用状況を視覚的に把握。

## 📱 ブラウザ対応・既知の制約

### PWA インストール

| ブラウザ | PWAインストール | ホーム画面追加 |
|---|---|---|
| Android Chrome | ✅ | ✅ |
| Android Edge | ✅ | ✅ |
| iOS Safari | ✅ | ✅ |

### Web Share Target（他アプリからの画像共有）

他のアプリから「共有」でおしこよに画像を直接送る機能です。

| ブラウザ | 対応状況 |
|---|---|
| Android Chrome | ✅ 対応 |
| Android Edge | ✗ 非対応（ブラウザの制限） |

> [!NOTE]
> Android Edge は PWA としての正常インストールには対応していますが、Web Share Target API（他アプリからの画像共有）を Android の Intent システムに登録しないため、共有先一覧に表示されません。これはアプリ側では解決できないブラウザの制限です。Edge をご利用の場合は、設定画面の「画像を追加」ボタンからファイルを直接選択してください。

## 📖 使い方 (Quick Start)

1. **アプリを開く**: [デモサイト](https://daraddara.github.io/oshikoyo/)へアクセス。
2. **自分好みに設定**: 右上の ⚙️ アイコンから、画像の表示方法やカレンダーをカスタマイズ。
3. **記念日を登録**: あなただけの「推し事」スケジュールをカレンダーに反映。
