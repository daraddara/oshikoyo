# 🗓️ Oshigoto-Calendar (推しカレ)

**「推し」との日常を、もっと美しく、もっと身近に。**

Oshigoto-Calendar（推しカレ）は、あなたの大切な「推し」の予定や記念日を管理するためにデザインされた、洗練されたモダンカレンダーアプリケーションです。

## ✨ 魅力的な機能

### 🖼️ 推し画像表示
- ローカル画像の登録、ランダム表示、サイクル表示。
- **画像の切り替え間隔をプリセット（10秒〜1時間、または毎日0:00/4:00）で簡単に設定可能。**
- 画像エリアからのダイレクト操作（表示モード切替、画像送り・戻し）。
- ドラッグによる配置変更とリサイズ。
- **直感的なUI操作**: ドラッグ＆ドロップやスプリッターを使用して、直感的に画像やカレンダーの配置・サイズ変更を行えます。

### 🎨 あなた色のカレンダー
- **カスタム休日設定**: 「推し」の誕生日やデビュー日、大切な記念日を特別な色で彩ります。
- **ダークモード対応**: 夜の推し活も快適に。OSの設定に合わせて、美しく落ち着いたテーマに切り替わります。
- **ガラスモーフィズム**: 最新のデザイントレンドを取り入れた、透明感のあるインターフェース。

### 🛠️ 自由自在な表示
- **マルチ月表示**: 1ヶ月から最大2ヶ月先まで。先の予定も一目瞭然。
- **ダイレクトなレイアウト切替**: メイン画面上のボタンからダイレクトにカレンダーの並びを縦・横に切り替えられます。
- **PWA対応**: PCやスマホのホーム画面に追加して、アプリ感覚ですぐに起動。

## 📖 使い方 (Quick Start)

1. **アプリを開く**: 上記のリンクからアクセス。
2. **自分好みに設定**: 右上の ⚙️ アイコンから、画像の表示方法やカレンダーのレイアウトをカスタマイズ。
3. **記念日を登録**: あなただけの「推し事」スケジュールをカレンダーに反映。
