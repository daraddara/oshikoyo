## ブラウザ起因の既知制約

### Android Edge — Web Share Target 非対応

| 項目 | 内容 |
|---|---|
| 現象 | Android Edge にインストールした PWA が、他アプリの共有先一覧に表示されない |
| 原因 | Edge for Android が Web Share Target API を Android の Intent システムに登録しない（ブラウザ未実装） |
| 影響範囲 | Android Edge ユーザーのみ。Chrome インストール時は正常動作 |
| アプリ側の対応 | 設定画面の「画像を追加」ボタンによるファイル選択で代替可能 |
| コード側の対応 | 不要（manifest・SW の実装は正しい。ブラウザ制限のため修正不可） |
