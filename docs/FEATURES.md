# 実装済み機能カタログ (Features)

おしこよの全実装済み機能を分類別にまとめたリファレンスです。

---

## カレンダー

| 機能 | 概要 |
|:---|:---|
| 月別カレンダー表示 | 1〜2ヶ月分を縦・横レイアウトで表示 |
| 日本祝日自動表示 | 振替休日・特例措置を含む祝日計算 |
| 週の開始日設定 | 日曜始まり / 月曜始まりを選択可能 |
| 月ナビゲーション | 前月・翌月ボタン、今日ボタンでリセット |
| 2ヶ月表示トグル | ヘッダーのボタンで1ヶ月⇄2ヶ月を切り替え |
| 日付詳細ポップアップ | デスクトップ: ホバーポップアップ / モバイル: ボトムシート |

## 推し管理

| 機能 | 概要 |
|:---|:---|
| 推しの追加・編集・削除 | 名前・カラー・記念日・タグを管理 |
| 記念日タイプ | 誕生日・デビュー記念日のビルトインタイプ＋カスタムタイプ |
| カスタムイベントタイプ | アイコン付きのユーザー定義タイプを追加・削除 |
| 推しカラー | カレンダーセルの記念日表示色として使用 |
| 推し一覧テーブル | 検索・ソート・色コーディング |
| CSVインポート | 推し情報・記念日を一括登録（重複検出あり） |
| CSVテンプレートDL | サンプルCSVをダウンロードして編集 |
| 次回記念日計算 | 最も近い記念日を自動計算して表示 |

## 画像・メディア

### 登録方法

| 方法 | 対応環境 |
|:---|:---|
| ファイル選択 | 全環境 |
| フォルダ一括登録 | デスクトップ Chrome / Edge |
| クリップボード貼り付け | Chrome / Edge |
| ドラッグ＆ドロップ | デスクトップ |
| Web Share Target | Android Chrome のみ（他アプリの「共有」から直接送信） |

### 表示モード

| モード | 動作 |
|:---|:---|
| 固定（ピン留め） | 特定の画像を固定表示。ホームタブにピンアイコン表示 |
| ランダム | 設定間隔ごとにランダムに切り替え |
| サイクル | 登録順に順番に切り替え |

### 切り替え間隔

10秒 / 30秒 / 1分 / 10分 / 1時間 / 毎日 0:00 / 毎日 4:00 / 起動時のみ

固定モードを解除した際は、最後に使用していた間隔を自動復元。

### レイアウト

| レイアウト | 配置 |
|:---|:---|
| スマート（自動） | 縦長画像→左配置・横長画像→上配置を自動判定 |
| 上 / 下 / 左 / 右 | 手動固定 |
| 没入モード | 画像フルスクリーン、UIはマウスオーバー時のみ表示 |

### その他

| 機能 | 概要 |
|:---|:---|
| プレビューモーダル | 追加前に確認・タグ付け |
| ライトボックスビューアー | 拡大表示・削除・タグ編集 |
| スプリッター | 画像/カレンダーの境界をドラッグしてサイズ調整 |
| デフォルト画像 | 初回起動時に横長・縦長サンプル画像を自動登録 |

## タグシステム

| 機能 | 概要 |
|:---|:---|
| 画像タグ | 各画像に複数タグを付与 |
| 推しタグ | 各推しに複数タグを付与 |
| タグオートコンプリート | 既存タグを候補表示 |
| 記念日×タグ連動 | 記念日に対応する推しのタグ付き画像を優先/専有表示 |
| 記念日表示モード | **優先**: 80%の確率でタグ一致画像 / **専有**: タグ一致画像のみ |
| タグフィルタリング | 画像管理画面でタグで絞り込み |
| タグマージ | インポート時に文字列照合で既存タグとマージ |

## ストレージ・バックアップ

### 画像自動圧縮

| 設定 | 内容 |
|:---|:---|
| 圧縮なし | 元ファイルのまま保存 |
| 標準（推奨） | 長辺最大2560px、JPEG品質88% |
| 積極的 | 長辺最大1920px、JPEG品質78% |

GIFは対象外。透過PNGは白背景に変換。

### バックアップ形式

| 形式 | 対象 | 復元モード |
|:---|:---|:---|
| フルバックアップ | 設定・推し・画像・タグ・メタデータ全て | **完全置換**（既存データ削除） |
| 画像+タグパッケージ | 画像とタグ情報のみ | **マージ**（既存データを保持して追加） |

### その他

- ストレージ使用量インジケーター（使用MB・上限MB・画像枚数）
- 登録済み画像の一括圧縮
- 全画像削除

## モバイル専用UI

### ボトムタブナビゲーション

| タブ | 内容 |
|:---|:---|
| ホーム | カレンダー・推し画像メイン画面。タップでポップオーバーを表示 |
| 画像ライブラリ | 登録画像のグリッド表示・管理 |
| 推し管理 | 推し一覧・追加・編集 |
| 設定 | 6つのサブパネルに整理された設定 |

### 再生モードポップオーバー

ホームタブをタップすると表示。表示モード（固定/ランダム/サイクル）と切り替え間隔を変更できます。固定モード中はホームタブにピンアイコンが表示されます。

### 推し管理パネル

- 仮想スクロール（大量の推しでも高速）
- かな行グループラベル（あ/か/さ...行）
- 右端インデックスレール（行頭への素早いジャンプ）
- スクロールバブル（現在位置インジケーター）

### モバイル設定サブパネル

| パネル | 設定項目 |
|:---|:---|
| 全般 | 週の開始日、レイアウトリセット |
| 記念日 | 画像表示モード（優先/専有） |
| 画像とストレージ | 圧縮設定・画像追加・ストレージ管理・バックアップ |
| イベント管理 | カスタムイベントタイプの追加・削除 |
| データ | フルバックアップ・復元 |
| アプリ情報 | バージョン確認・アップデート確認 |

### その他モバイル機能

- スワイプジェスチャーによる月移動
- ボトムシートによる日付詳細表示
- ティッカーバー（当日・直近の記念日をスクロール表示）
- 自動バー非表示（操作後一定時間でボトムバーを隠す）

## PWA・システム

| 機能 | 概要 |
|:---|:---|
| PWAインストール | Android Chrome / Android Edge / iOS Safari でホーム画面追加 |
| オフライン対応 | Service Worker によるキャッシュ |
| Web Share Target | Android Chrome からの画像直接受信 |
| 自動アップデート検知 | Service Worker の更新を検知してトースト通知 |
| PWAデバッグパネル | 設定タブを7回連続タップで起動（開発者向け） |

## UXユーティリティ

| 機能 | 概要 |
|:---|:---|
| トースト通知 | 操作結果を画面下部に一時表示 |
| カスタム確認ダイアログ | `window.confirm()` に代わるデザイン統一のモーダル |
| ダークモード | OS設定に自動連動（手動スイッチは未実装） |
| ガラスモーフィズムUI | backdrop-filter による半透明デザイン |
| XSS対策 | `escapeHTML()` による全出力エスケープ |
| カラーコントラスト自動調整 | 推しカラーの背景に対して黒/白テキストを自動選択 |
