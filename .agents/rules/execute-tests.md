# 手順
1. **環境確認**: `node_modules` が存在するか確認し、なければ `npm install` を実行してください。
2. **テスト実行**: ターミナルで `npm test`（Vitest）を実行してください。
3. **結果の分析**:
   - 失敗した場合：`logic.test.js` で抽出している `script.js` のロジックを確認し、バグを修正してください。
   - 修正後、再度 `npm test` を実行して合格を確認してください。
4. **報告**: 実行結果と、パスしたテストケースの一覧を日本語で報告してください。
