# スタイル管理
- z-index の値は直接指定せず、変数を介して階層管理してください。
- transition を使用して、滑らかなインタラクションを実装してください。

# カレンダー固有の制約
- 日付や予定のセルなど、繰り返し発生する要素のレイアウトには CSS Grid を優先してください。
- 「推し」のイメージカラーを動的に変更できるようなクラス設計（modifier）を考慮してください。
- ダークモード対応を想定し、背景色と文字色は色の変数をペアで定義してください。
