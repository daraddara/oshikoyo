# 言語と言語形式の厳格な指定
- **チャット応答だけでなく、生成されるすべてのファイルコンテンツ（Implementation Plan, Walkthrough, 修正案など）の内部テキストを必ず日本語で構成してください。**
- 英語の標準テンプレート（"Proposed Changes", "Test Suite Refactoring Plan" 等）は使用せず、これらもすべて日本語（例：「提案される変更」「テストスイートのリファクタリング計画」）に翻訳して記述してください。
- 専門用語を除き、ドキュメント内の見出し、箇条書き、説明文のすべてに日本語を適用してください。
