# 手順
1. **最終テスト**: `npm test`（Vitest）を実行し、すべてのテストがパスすることを確認してください。
2. **コミット**: 
   - 変更内容をステージングしてください。
   - `DEVELOPMENT.md` の指針に従い、変更内容を日本語で明確に記述したコミットメッセージを作成し、コミットしてください。
3. **最新化**: `main` ブランチに切り替え、最新の状態をプルしてください。
4. **マージ**: 作業ブランチを `main` にマージしてください。
5. **後処理**: マージが成功したら、ローカルの作業用ブランチを削除してください。
6. **完了報告**: 「マージが完了しました。最新のコードは main ブランチに反映されています」と日本語で報告してください。
