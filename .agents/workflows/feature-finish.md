# 手順
1. **最終確認**: `npm test` を実行し、すべてのテストがパスすることを確認してください。
2. **コミット**: 変更内容をステージングし、ルールに従ったメッセージでコミットしてください。
3. **統合準備**: `main` ブランチに切り替え、最新をプルしてください。
4. **マージ**: 作業ブランチを `main` にマージ（`git merge --no-ff`）してください。
5. **後片付け**: マージが成功したら、作業用ローカルブランチを削除してください。
6. 全行程の完了を報告してください。
