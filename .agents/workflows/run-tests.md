# ワークフロー手順
1. ターミナルで `npm test`（または `vitest run`）を実行して、現在のテスト状況を確認してください。
2. もしテストが失敗した場合は、エラーログを分析して原因を特定してください。
3. 失敗した原因がコードのバグであれば修正案を提示し、テストコード側の不備であればテストを修正してください。
4. すべてのテストがパスするまで、必要に応じて修正と再実行を繰り返してください。
5. 最後に、テストカバレッジの概要を日本語で報告してください。
