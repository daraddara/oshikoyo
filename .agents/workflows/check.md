# 手順
1. **サーバー起動**: ローカルサーバー（`npx serve .` 等）を起動し、ブラウザでアプリを開いてください。
2. **ビジュアル確認**:
   - ダークモードとライトモードを切り替え、すべてのテキストが視認可能か確認してください。
   - 画面幅を変更し、レスポンシブ表示（特にカレンダーのグリッド）が崩れていないか確認してください。
3. **機能動作確認**:
   - 設定パネルを開き、週の開始日や表示月数を変更して「保存」が反映されるか確認してください。
   - カレンダーの「前月/次月」ボタンが正しく動作するか確認してください。
4. **課題照合**: `ISSUES.md` に記載されている「モバイル表示の最適化 (A-04)」などの懸念点が解消されているか確認してください。
5. **報告**: 確認した項目と、スクリーンショットに基づいた結果を日本語で報告してください。
