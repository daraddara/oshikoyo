# 手順
1. **最終確認**: `npm test` を実行し、すべてのテストがパスすることを確認してください。
2. **最新の取り込み**: `main` ブランチの最新状態を作業ブランチに **Rebase（リベース）** して、競合がないか確認してください。
3. **統合（Squash Merge）**: 
   - `main` ブランチに切り替え、`git merge --squash [作業ブランチ名]` を実行してください。
   - 変更内容を一つにまとめてコミットし、コミットメッセージは日本語で要約を記述してください。
4. **後片付け**: ローカルの作業用ブランチを削除してください。
5. **プロセスの完全終了**:
   - **開発用サーバーやテスト用ブラウザが残っている場合は、すべて終了させてからタスクを完了してください。**
6. **完了報告**: 「マージを完了し、ブラウザを含むすべての開発プロセスを終了しました」と日本語で報告してください。
