# 手順
1. **最終確認**: `npm test` を実行し、すべてのテストがパスすることを確認してください。
2. **最新の取り込み**: `main` ブランチの最新状態を作業ブランチに **Rebase（リベース）** して、競合がないか確認してください。
3. **統合（Squash Merge）**: 
   - `main` ブランチに切り替えます。
   - `git merge --squash [作業ブランチ名]` を実行し、変更内容を一つにまとめてコミットしてください。
   - コミットメッセージは日本語で、変更内容の要約を記述してください。
4. **後片付け**: ローカルの作業用ブランチを削除してください。
5. **完了報告**: 「マージコミットを作らず、クリーンな履歴で main に統合しました」と報告してください。
